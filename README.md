# Proyecto Scrum - Registro de Asistencia

## Descripción del Proyecto
Esta es una aplicación web dinámica diseñada para facilitar y agilizar el control de asistencia de alumnos. El sistema permite al docente llevar un conteo exacto y organizado de la presencia en el aula, categorizando a cada estudiante según su puntualidad y gestionando toda la información de manera local y eficiente.

## Funcionalidades Principales
El proyecto se ha estructurado en distintos módulos clave para cumplir con los objetivos del marco de trabajo Scrum:

1. **Sistema de Autenticación (Login)**  
   Acceso seguro para docentes y administradores, garantizando que solo el personal autorizado pueda gestionar la información.

2. **Panel de Inicio y "Sobre Nosotros"**  
   Secciones informativas que explican el propósito del sistema, su funcionamiento y facilitan la navegación inicial del usuario.

3. **Formulario del Docente**  
   Interfaz intuitiva para el registro detallado de datos, incluyendo nombre del estudiante, curso, hora de ingreso, hora de salida y estado de asistencia.

4. **Filtrado por Estado**  
   Sistema que permite segmentar la lista de alumnos para identificar rápidamente quiénes están presentes, tienen tardanza o están ausentes.

5. **Contador de Asistencias**  
   Funcionalidad automatizada que contabiliza en tiempo real cuántos alumnos han sido registrados en la sesión actual.

6. **Sección de Contáctanos**  
   Espacio dedicado para que los usuarios puedan enviar consultas, reportar problemas o solicitar soporte técnico.

7. **Persistencia con LocalStorage**  
   El sistema funciona sin base de datos externa, utilizando el almacenamiento local del navegador para garantizar que los datos se mantengan guardados de manera segura en el equipo.
